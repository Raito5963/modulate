# Modulate プロジェクト

モジュラー・エルゴノミックキーボード「Modulate」のWebプロジェクトです。

## 📁 プロジェクト構成

- `landing-page/` - Next.js製のランディングページ（日本語・英語対応）
- `中間発表.txt` - プロジェクト概要（中間発表の内容）

## 🚀 クイックスタート

### ランディングページの起動

```bash
cd landing-page
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開きます。

詳細な設定については `landing-page/README.md` を参照してください。

## 🎯 プロジェクトの目的

タイピングによる手首や指の痛みを軽減し、作業効率を向上させる次世代キーボードの開発。

### 主な特徴

1. **エルゴノミクスデザイン** - 縦分割タイル型で自然な姿勢
2. **モジュラー設計** - 作業形態に合わせてカスタマイズ可能
3. **ワイヤレス接続** - Bluetooth対応で携帯性向上

### ターゲット

- プログラマー
- デザイナー
- ライター
- RSI/CTSのリスクを抱える長時間タイピングユーザー

## 📊 市場背景

- タイピング中に痛みを感じる人: **50%**
- 作業効率が悪いと感じる人: **12.5%**
- RSI/CTSなどの疾患リスクは増加傾向
