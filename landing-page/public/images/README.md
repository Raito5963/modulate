## 画像フォーマット

- **推奨**: WebP（最適化済み）
- **代替**: JPG（写真）、PNG（透過が必要な場合）
- **品質**: 高品質だが、ウェブ最適化済み（各ファイル500KB以下推奨）

## 画像を配置したら

各コンポーネントファイル内のプレースホルダーを実際の画像に置き換えてください:

```tsx
// 例: HeroSection.tsx
import Image from 'next/image';

// プレースホルダーのdivを削除して
<Image
  src="/images/hero-background.jpg"
  alt="Modulate Keyboard Hero"
  fill
  className="object-cover"
  priority
/>
```

## 最適化のヒント

1. **画像圧縮**: TinyPNG、Squoosh、ImageOptimなどで圧縮
2. **Next.js Image**: `next/image`を使用すると自動最適化
3. **遅延読み込み**: 重要でない画像は`loading="lazy"`を設定
4. **WebP変換**: 可能な限りWebPフォーマットを使用
